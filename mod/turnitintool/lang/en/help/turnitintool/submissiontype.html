Turnitin Assignments - Submission Type

Indicates the submission type(s) you are allowed to submit to Turnitin. The options are:

  • File Upload
    You may upload the following file types to the File Upload submission type: doc, docx, pdf, rtf, txt and html. When uploading a PDF file you must ensure that the text content is selectable as opposed to the documents text represented by and image.

  • Text Submission
    You may use the Text Submission to copy and paste text content to Turnitin.
