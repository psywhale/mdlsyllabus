Trabalhos Turnitin - Tipos de Envios

Os envios podem ser feitos em três formatos diferentes.

  • Upload do Arquivo

    Permite ao usuário fazer o upload de arquivo dos seguintes tipos:

    • Microsoft Word (doc,docx,rtf,html)
    • PDF (pdf)
    • Texto Simples (txt)

  • Envio de Texto

    Permite ao usuário copiar e colar texto simples dentro da área de texto para fazer o upload para o Turnitin.
