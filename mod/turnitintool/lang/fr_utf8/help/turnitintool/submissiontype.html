Exercices Turnitin - Méthode d’envoi

Indique les types de fichier qui peuvent être soumis à Turnitin. Les options sont :

  • Téléchargement d’un document
    Vous pouvez télécharger les types de fichier suivants Téléchargement d’un document doc, docx, pdf, rtf, txt et html. Quand vous téléchargez un document en PDF vous devez vous assurer que le texte du document peut être sélectionné. Le texte ne peut pas être sous forme d’image.

  • Envoi de texte
    Vous pouvez utiliser l\' Envoi de document pour copier et coller le contenu d’un texte dans Turnitin.
