Ejercicios Turnitin - Parte del Ejercicio

Determina si las partes del ejercicio son partes nombradas o si este ejercicio es un ejercicio del portafolio

  • Partes Nombradas

    Esta configuración le permite establecer un nombre para la parte y el número de anotaciones permitidas para cada parte del ejercicio. También le permite establecer individualmente las fechas de inicio, de entrega y de publicación para cada parte del ejercicio.

  • Portafolio

    Esta configuración le permite agrupar las partes del ejercicio en un solo ejercicio. Se asignan automáticamente las fechas de inicio, de entrega y de publicación generales para cada parte. Las partes no son nombradas y los estudiantes entregan un número determinado de trabajos. La nota general del ejercicio se calcula usando el promedio de las notas asignadas para cada entrega.
