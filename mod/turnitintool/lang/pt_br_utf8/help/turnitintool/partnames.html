Trabalhos Turnitin - Partes do Trabalho

Determina se as Partes desse trabalho são Partes Nomeadas ou se esse trabalho é um trabalho do Portfolio.

  • Partes Nomeadas

    Essa configuração permite que você defina um nome da parte e o número de correções atribuídas a cada parte do trabalho. Também permite que você defina individualmente as datas de início, de entrega e de publicação para cada parte do trabalho.

  • Portfolio

    Essa configuração permite que você agrupe as partes do trabalho em um trabalho. A cada parte é atribuída automaticamente as datas de início, de entrega e de publicação do trabalho geral padrão. As partes não são nomeadas e os usuários enviam um determinado número de envios. A nota geral para o trabalho é calculada usando uma média de notas dadas para cada envio.
