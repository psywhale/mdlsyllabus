Exercices Turnitin - Types d’envoi

Trois formats différents peuvent être soumis

  • Téléchargement du document

    Les utilisateurs peuvent télécharger les formats de documents suivants :

    • Microsoft Word (doc,docx,rtf,html)
    • PDF (pdf)
    • Texte brut

  • Envoi texte

    permet aux utilisateurs de copier et de coller un texte dans la zone de texte et de le télécharger vers le serveur Turnitin
