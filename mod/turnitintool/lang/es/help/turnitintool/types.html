Ejercicios Turnitin - Tipos de entrega

Las entregas se pueden hacer en tres formatos diferentes.

  • Carga de archivos

    Permite al usuario cargar archivos de formato:

    • Microsoft Word (doc,docx,rtf,html)
    • PDF (pdf)
    • Texto sin formato (txt)

  • Entrega de texto

    Permite al usuario copiar y pegar texto sin formato al cuadro de texto para cargarlo a Turnitin.
